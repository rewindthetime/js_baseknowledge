## ⚠️ Important: Node.js Only

This repository is designed **exclusively for Node.js**.

That means:

* ❌ Not intended for browser use
* ❌ No frontend (HTML/CSS/DOM)
* ✅ Focus on backend logic
* ✅ Run using `node` in the terminal
* ✅ It works in browser yes but i didn't tell you how to get something in html (probably coming soon)

### Why?

Node.js allows JavaScript to run outside of the browser. This repository takes advantage of that environment to demonstrate core JavaScript concepts in a server-side context.

---

## 🧠 What does this repository do?

This repository is a **learning collection for JavaScript fundamentals**.

It helps you:

* understand basic syntax
* learn how functions work
* work with variables
* understand arrays and objects
* explore common Node.js patterns

In short:

> A collection of simple examples to learn JavaScript from scratch specifically in a Node.js environment.

---

## 📁 Example Content

You can expect code like this:

```js
function greet(name) {
    console.log(`Hello ${name}`);
}

greet("World");
```

---

## 🎯 Goal of the Repository

The main goal is to provide a **simple, clean, and beginner-friendly introduction** to JavaScript using Node.js.

It is especially useful for:

* beginners starting with JavaScript
* people who want to understand backend basics
* quick reference for core concepts

## 🚀 How to Run

1. Install Node.js
2. Clone the repository
3. Run a file:

```bash
cd (to_the_folder)
node .
```
